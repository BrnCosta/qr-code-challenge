### What I learned

Using css variables:

```css

    --white: hsl(0, 0%, 100%);
    --lightGray: hsl(212, 45%, 89%);
    --grayishBlue: hsl(220, 15%, 55%);
    --darkBlue: hsl(218, 44%, 22%);
}

.info-text p {
    color: var(--darkBlue);
}
```
